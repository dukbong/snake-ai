# Snake AI 프로젝트

## 프로젝트 개요
PPO(Proximal Policy Optimization) 알고리즘을 사용한 Snake 게임 AI 학습 프로젝트.

## 핵심 규칙

### 절대 금지: 거리 기반 보상(Distance-Based Reward Shaping)
**거리 기반 보상을 절대 추가하지 말 것.**

- 음식과의 거리가 가까워질수록 보상, 멀어질수록 패널티를 주는 방식은 사용 금지
- 이유: 뱀이 길어질수록 몸통을 피해야 하는 복잡한 경로가 필요한데, 거리 기반 보상이 이를 방해함
- 결과적으로 짧은 뱀일 때는 잘 작동하지만 긴 뱀에서는 오히려 성능이 저하됨

## 현재 보상 체계 (Sparse Reward)
```
+1  : 음식 먹기
-1  : 죽기 (벽 충돌 또는 몸통 충돌)
 0  : 그 외 모든 상황
```

이 sparse reward 구조를 유지할 것. 추가적인 reward shaping은 도입하지 말 것.
